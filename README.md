# Control II

Repositorio académico donde se almacenan los diferentes talleres desarrollados durante el curso de **Control **.

El objetivo de este repositorio es documentar de forma organizada los ejercicios, análisis, simulaciones y resultados obtenidos en los distintos talleres del curso.

## Contenido del repositorio

Cada carpeta corresponde a un taller independiente e incluye:

- Enunciado del problema
- Scripts desarrollados (principalmente en MATLAB)
- Informe 


## Objetivos

Este repositorio busca:

- Mantener un registro organizado del trabajo académico
- Documentar los procesos de diseño y análisis en control
- Facilitar la reproducibilidad de los resultados
- Servir como material de referencia para futuros estudios

## Herramientas utilizadas

Las principales herramientas utilizadas en los talleres son:

- MATLAB
- Control System Toolbox
- System Identification Toolbox
- Simulaciones de sistemas dinámicos



## Notas

Este repositorio tiene fines académicos y de aprendizaje.
